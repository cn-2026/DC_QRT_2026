# DC_QRT_2026

Projet de Data Challenge QRT 2026 pour apprentissage statistique.

Un espace de stockage de données brutes et preprocessed est présent dans le dossier "data".

Les fonctions du projet doivent être placées dans le "main" dans les dossiers associés à leurs fonctionnalités.

Pour la rédaction de théorie et présentation de résultats un notebook est présent.
